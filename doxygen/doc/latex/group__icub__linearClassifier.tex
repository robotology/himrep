The module implements a linear classifier wrapped around liblinear library. 

\hypertarget{group__GIECoder_intro_sec}{}\doxysubsection{Description}\label{group__GIECoder_intro_sec}
This module is responsible for learning and classify feature vectors. Input features are passed, the output are the scores of SVM machines. The module provides training methods with real-\/time performances.

The commands sent as bottles to the module port /linear\+Classifier/rpc are the following\+:

(notation\+: \mbox{[}.\mbox{]} identifies a vocab, $<$.$>$ specifies a double, \char`\"{}.\char`\"{} specifies a string)

{\bfseries{SAVE}} ~\newline
format\+: \mbox{[}save\mbox{]} \char`\"{}class\+\_\+name\char`\"{} ~\newline
action\+: starts to dump feature vector in the \char`\"{}class\+\_\+name\char`\"{} folder.

{\bfseries{STOP}} ~\newline
format\+: \mbox{[}stop\mbox{]} ~\newline
action\+: stops all the current activities, the module will be waiting for new commands.

{\bfseries{TRAIN}} ~\newline
format\+: \mbox{[}train\mbox{]} ~\newline
action\+: starts the training process by reading all the database previously acquired via the \mbox{[}save\mbox{]} command.

{\bfseries{FORGET}} ~\newline
format\+: \mbox{[}forget\mbox{]} \char`\"{}class\char`\"{} ~\newline
action\+: forgets the \char`\"{}class\char`\"{}, deleting all the feature vectors in the database. If \char`\"{}class\char`\"{}=\char`\"{}all\char`\"{} all the classes are forgotten.

{\bfseries{LIST}} ~\newline
format\+: \mbox{[}obj\+List\mbox{]} ~\newline
action\+: retrieves the current classes saved in the database.

{\bfseries{CHANGE NAME}} ~\newline
format\+: \mbox{[}change\+Name\mbox{]} \char`\"{}old\+\_\+name\char`\"{} \char`\"{}new\+\_\+name\char`\"{} ~\newline
action\+: change the current name of a known object with a new name.

format\+: \mbox{[}recognize\mbox{]} ~\newline
